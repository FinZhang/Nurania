项目规格说明书：诺拉尼亚（Nurania）世界观展示站
1. 核心愿景 (The Vibe)
为中世纪西幻背景的世界观诺拉尼亚大陆制作展示站网页，设定上其展示的内容是该世界观中一位学者撰写的大陆志。要求完全适配电脑宽屏和移动端窄屏。
风格关键词： 史书背景，史诗感、神秘主义、羊皮纸质感与现代极简 UI 的结合（类似《艾尔登法环》或《巫师 3》的 UI 风格）。
视觉调性： 书籍纸质背景，文字使用带有精致的手写感的字体。交互时要有微光（Glow）和淡入淡出（Fade）效果。

2. 技术栈建议
框架： Next.js (App Router)
样式： Tailwind CSS
动画： Framer Motion (用于平滑的页面转场和元素浮现)
图标： Lucide React (作为基础，必要时由 AI 生成 SVG 纹章)

3. 站点地图与功能逻辑
3.1 门户主页 (Index / Hero)
内容： 左侧是一本符合世界观风格的精致书籍的封面：标题为诺拉尼亚行思录 Under the Nuranian Skies: A Compendium（中英双语），右侧是世界观地图
交互： 点击书籍后出现书籍展开的动画，进入目录页
氛围： 史诗感的风格

3.2 目录页
内容：当前展示站所有页面，按层级列出网站所有文章条目
交互：点击条目后，跳转至对应的文章内容页，点击返回会回到门户主页

3.3 文章内容页
内容：左侧边栏为导航（显示目录），右侧为文章主页面
交互：点击导航内其他条目，跳转到对应文章内容页，或者点击返回会回到目录页。

4. 开发者指令 
全局样式： 在 globals.css 中定义一套符合“奇幻史诗”的主题色板。
组件化： 所有的导航栏、页脚、内容页必须作为独立组件开发。
响应式： 必须完美适配移动端，自动排版。
数据驱动： 所有的文章内容页内容请预留 data/world_content.ts 文件，方便我后续填充具体细节。
