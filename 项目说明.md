# 诺拉尼亚（Nurania）项目说明

索引版：站点整体约定与快速定位入口；具体实现细节已拆分到 `siteDocu/` 中。

---

## 1. 核心愿景与风格

- **定位**：中世纪西幻「诺拉尼亚大陆」展示站；品牌为「尤里梅尔全集」。
- **适配**：电脑宽屏与移动端窄屏。
- **风格**：史书背景、史诗感、神秘主义、羊皮纸质感与现代极简 UI。
- **视觉**：书籍纸质背景、手写感字体；交互带微光（Glow）与淡入淡出（Fade）。

---

## 2. 技术栈

- **框架**：Next.js（App Router）
- **样式**：Tailwind CSS
- **动画**：Framer Motion
- **图标**：Lucide React
- **构建**：生产环境 `output: "export"` 静态导出；开发环境不导出以放宽 `generateStaticParams` 校验。

---

## 3. 路由与站点结构

- **basePath**：`/Nurania`，所有资源与链接均带此前缀（见 `next.config.ts`、`image-loader.ts`）。
- **路由约定**：
  - `/`：**书架**（主页），展示多本书封面，点击某书进入该书目录。
  - `/[bookSlug]/toc`：某书的目录页（如 `/compendium/toc`、`/upTheRiver/toc`、`/deathOfDevout/toc`）。
  - `/[bookSlug]/article/[...slug]`：某书的文章页（如 `/compendium/article/作者序`）。
- **可扩展性**：新增书籍只需在 `src/lib/books.ts` 的 `BOOKS` 数组中增加一项，并在 `data/`、`public/` 下增加对应平级目录即可，无需改页面组件。

---

## 4. 书籍配置与数据/资源目录

- **配置入口**：`src/lib/books.ts`。每本书包含：
  - `slug`：路由与标识（如 `compendium`、`upTheRiver`、`deathOfDevout`）
  - `title` / `titleEn`：中英文书名
  - `cover`：封面图路径（如 `/compendium/book_cover.webp`），当前参考封面尺寸标准：900x1200
  - `intro`：目录页简介区块正文
  - `dataDir`：该书 Markdown 数据目录（相对项目根，如 `data/compendium`）
  - `publicDir`：该书图片在 `public` 下的文件夹名（如 `compendium`）
  - `footerSubline`（可选）：该书内页脚左侧副标题

- **数据目录**：每本书在 `data/` 下占一个**平级**文件夹，彼此不嵌套。
  - 例：`data/compendium/`（第一本）、`data/upTheRiver/`（第二本）、`data/deathOfDevout/`（第三本）。
  - 每书目录内为层级 Markdown 与 `_order.json`，由 `src/lib/content.ts` 按 `bookSlug` 读取对应 `dataDir`。

- **图片目录**：每本书在 `public/` 下占一个**平级**文件夹。
  - 例：`public/compendium/`（第一本封面、世界地图、诸国列志等）、`public/upTheRiver/`（第二本封面等）、`public/deathOfDevout/`（第三本封面等）。
  - 引用时路径如 `/compendium/book_cover.webp`、`/upTheRiver/book_cover.webp`、`/deathOfDevout/book_cover.webp`。世界地图为 `compendium/world_map.webp`、`compendium/world_map_labelled.webp`；题图推荐使用 WebP。
